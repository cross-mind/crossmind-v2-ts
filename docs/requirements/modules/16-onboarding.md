## 1. 模块概述

### 1.1 模块定位

**核心价值**: 降低新用户学习曲线，在 10 分钟内体验核心价值。

**目标**:
- 新用户理解 CrossMind 是什么
- 完成首次完整流程（想法 → Agent → 任务）
- 留存率 > 60%

### 1.2 优先级

- **P0**: 欢迎流程、示例数据、免费 Agent 体验
- **P1**: 分步引导、进度追踪
- **P2**: 个性化引导

---

## 2. 核心用户场景

### 场景 2.1: 首次登录

**基本流程**:
```
用户完成 OAuth 登录
    ↓
显示欢迎页："欢迎来到 CrossMind"
    ↓
快速介绍三大模块（30 秒动画）
    ↓
选择："从示例开始" 或 "创建空白项目"
    ↓
[选择示例] 进入预填充的示例项目
    ↓
分步引导开始
```

### 场景 2.2: 体验完整流程

**引导步骤**:
```
步骤 1: 查看 Canvas 上的示例想法节点
    ↓
步骤 2: 点击节点，与 AI 对话
    ↓
步骤 3: 雇佣第一个 Agent（免费）
    ↓
步骤 4: 查看 Agent 产出
    ↓
步骤 5: 将想法转化为任务
    ↓
完成引导 🎉
```

## 3. 功能详细设计

### 3.1 欢迎流程

#### 3.1.1 欢迎页

**内容**:
- 标题："欢迎来到 CrossMind"
- 副标题："从想法到产品，只需几天"
- 三大模块介绍（卡片形式）

**CTA**:
- "从示例开始"（推荐）
- "创建空白项目"

**优先级**: P0

#### 3.1.2 快速介绍

**形式**: 30 秒动画或交互演示

**内容**:
- Canvas：可视化组织想法
- 任务中心：追踪执行进度
- Agent 雇佣中心：AI 助你加速

**跳过**: 允许用户跳过

**优先级**: P0

### 3.2 示例数据

**预填充内容**:
- Canvas：5-8 个示例节点（SaaS 产品想法）
- 任务：3-5 个示例任务
- Agent 产出：1 个完成的调研报告

**用途**: 用户可直接体验所有功能

**优先级**: P0

### 3.3 分步引导

#### 3.3.1 引导步骤

| 步骤 | 操作 | 提示气泡 | 优先级 |
|------|------|---------|-------|
| 1 | 查看 Canvas 节点 | "这是你的想法节点，点击查看详情" | P0 |
| 2 | 与 AI 对话 | "点击 '✨ AI 辅助' 与 AI 聊聊" | P0 |
| 3 | 雇佣 Agent | "让 Agent 帮你做市场调研（首次免费）" | P0 |
| 4 | 查看产出 | "Agent 已完成，查看调研报告" | P0 |
| 5 | 创建任务 | "将想法转化为可执行任务" | P0 |

#### 3.3.2 引导形式

**提示气泡**:
- 箭头指向目标元素
- 简短文字说明（< 20 字）
- "下一步"按钮

**高亮**: 目标元素高亮，其他区域半透明遮罩

**跳过**: 允许随时退出引导

**优先级**: P1

### 3.4 免费首体验

**触发**: 引导步骤 3

**内容**: 第一次雇佣任意 Agent 免费

**提示**: "首次体验免费 🎉"

**优先级**: P0

### 3.5 完成庆祝

**触发**: 完成所有引导步骤

**展示**:
- 庆祝动画 🎉
- "你已掌握 CrossMind 基础！"
- 下一步建议："创建你的第一个真实项目"

**优先级**: P0

## 4. 交互流程设计

### 4.1 完整的 Onboarding 流程

```
[开始] 用户首次登录
    ↓
[欢迎页] 显示欢迎信息和快速介绍
    ↓
[选择] 用户点击"从示例开始"
    ↓
[创建] 系统创建示例项目并进入
    ↓
[步骤1] 提示气泡："这是你的想法节点"
    ↓
[用户] 点击节点查看
    ↓
[步骤2] 提示气泡："点击 AI 辅助"
    ↓
[用户] 与 AI 对话
    ↓
[步骤3] 提示气泡："雇佣 Reddit 调研 Agent（免费）"
    ↓
[用户] 下单 Agent
    ↓
[等待] Agent 执行（1-2 分钟，显示进度）
    ↓
[步骤4] 通知："Agent 完成了！查看报告"
    ↓
[用户] 查看产出
    ↓
[步骤5] 提示气泡："创建任务追踪进度"
    ↓
[用户] 创建任务
    ↓
[完成] 庆祝动画 🎉
    ↓
[引导] "创建你的第一个真实项目"
    ↓
[结束] 进入正常使用
```

**总时长**: < 10 分钟

## 5. 业务规则

### 5.1 触发条件

**谁触发**: 首次登录的新用户

**何时触发**: 登录后立即

**跳过**: 用户可选择跳过，但建议完成

### 5.2 进度保存

**规则**:
- 用户可随时退出
- 进度自动保存
- 下次登录继续
