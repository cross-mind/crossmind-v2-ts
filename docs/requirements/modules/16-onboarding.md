## 1. 模块概述

### 1.1 模块定位

**核心价值**: 降低新用户学习曲线，在 10 分钟内体验核心价值。

**目标**:
- 新用户理解 CrossMind 是什么
- 完成首次完整流程（想法 → Agent → 任务）
- 留存率 > 60%

### 1.2 优先级

- **P0**: 欢迎流程、示例数据、免费 Agent 体验
- **P1**: 分步引导、进度追踪
- **P2**: 个性化引导

---

## 2. 核心用户场景

### 场景 2.1: 首次登录

**基本流程**:
```
用户完成 OAuth 登录
    ↓
显示欢迎页："欢迎来到 CrossMind"
    ↓
快速介绍三大模块（30 秒动画）
    ↓
选择："从示例开始" 或 "创建空白项目"
    ↓
[选择示例] 进入预填充的示例项目
    ↓
分步引导开始
```

### 场景 2.2: 体验完整流程（MVP 精简为 3 步）

**引导步骤**:
```
步骤 1: 查看 Canvas 上的示例文档节点
    ↓
步骤 2: 雇佣第一个 Agent（免费）
    ↓
步骤 3: 创建任务追踪进度
    ↓
完成引导 🎉
```

**说明**: MVP 版本简化为 3 步核心流程，专注于「浏览 Canvas → 雇佣 Agent → 创建任务」的完整闭环，降低学习成本。

## 3. 功能详细设计

### 3.1 欢迎流程

#### 3.1.1 欢迎页

**内容**:
- 标题："欢迎来到 CrossMind"
- 副标题："从想法到产品，只需几天"
- 三大模块介绍（卡片形式）

**CTA**:
- "从示例开始"（推荐）
- "创建空白项目"

**优先级**: P0

#### 3.1.2 快速介绍

**形式**: 30 秒动画或交互演示

**内容**:
- Canvas：可视化组织想法
- 任务中心：追踪执行进度
- Agent 雇佣中心：AI 助你加速

**跳过**: 允许用户跳过

**优先级**: P0

### 3.2 示例数据（MVP 精简）

**预填充内容**:
- Canvas：2-3 个示例节点（SaaS 产品想法）
  - 1 个文档节点（产品想法）
  - 1 个想法节点（待验证的特性）
  - 1 个 Agent 节点（已完成的调研报告）
- 任务：1-2 个示例任务
- Agent 产出：1 个完成的 Reddit 调研报告

**用途**: 用户可快速理解核心概念，减少认知负担

**优先级**: P0

### 3.3 分步引导

#### 3.3.1 引导步骤（MVP 精简为 3 步）

| 步骤 | 操作 | 提示气泡 | 优先级 |
|------|------|---------|-------|
| 1 | 查看 Canvas 示例 | "这是你的项目画布，点击节点查看" | P0 |
| 2 | 雇佣 Agent | "雇佣 Reddit 调研 Agent（首次免费）" | P0 |
| 3 | 创建任务 | "从想法创建可执行任务" | P0 |

**移除步骤**（延后至 V1.0）:
- "与 AI 对话" - 用户可自行探索
- "查看产出" - 在步骤 2 雇佣后自然会查看

#### 3.3.2 引导形式

**提示气泡**:
- 箭头指向目标元素
- 简短文字说明（< 20 字）
- "下一步"按钮

**高亮**: 目标元素高亮，其他区域半透明遮罩

**跳过**: 允许随时退出引导

**优先级**: P1

### 3.4 免费首体验

**触发**: 引导步骤 3

**内容**: 第一次雇佣任意 Agent 免费

**提示**: "首次体验免费 🎉"

**优先级**: P0

### 3.5 完成庆祝

**触发**: 完成所有引导步骤

**展示**:
- 庆祝动画 🎉
- "你已掌握 CrossMind 基础！"
- 下一步建议："创建你的第一个真实项目"

**优先级**: P0

## 4. 交互流程设计

### 4.1 完整的 Onboarding 流程（MVP 精简版）

```
[开始] 用户首次登录
    ↓
[欢迎页] 显示欢迎信息和快速介绍（30秒）
    ↓
[选择] 用户点击"从示例开始"
    ↓
[创建] 系统创建示例项目并进入（包含2-3个节点）
    ↓
[步骤1] 提示气泡："这是你的项目画布，点击节点查看"
    ↓
[用户] 浏览示例节点
    ↓
[步骤2] 提示气泡："雇佣 Reddit 调研 Agent（首次免费）"
    ↓
[用户] 下单 Agent（填写调研关键词）
    ↓
[等待] Agent 执行（1-2 分钟，显示进度）
    ↓
[通知] "Agent 完成了！查看调研报告"
    ↓
[用户] 查看产出
    ↓
[步骤3] 提示气泡："从想法创建可执行任务"
    ↓
[用户] 创建任务
    ↓
[完成] 庆祝动画 🎉 "你已掌握 CrossMind 核心流程！"
    ↓
[引导] "创建你的第一个真实项目"
    ↓
[结束] 进入正常使用
```

**总时长**: < 5 分钟（不含 Agent 执行等待）
**完整时长**: < 10 分钟（含 Agent 执行）

## 5. 业务规则

### 5.1 触发条件

**谁触发**: 首次登录的新用户

**何时触发**: 登录后立即

**跳过**: 用户可选择跳过，但建议完成

### 5.2 进度保存

**规则**:
- 用户可随时退出
- 进度自动保存
- 下次登录继续
